# 🎬 Cinema App

A simple movie browsing web app built with **React** and **Vite**.

🔗 **Live Demo:** [cinema-ksjuh9ndn-mohammad-salim-s-projects.vercel.app](https://cinema-ksjuh9ndn-mohammad-salim-s-projects.vercel.app/)

## Pages

- **Home** – landing page
- **Movies** – browse movie listings
- **About** – project/about info
- **Contact** – contact section

## Tech Stack

- React
- Vite
- CSS / JavaScript

## Getting Started

```bash
# Clone the repo
git clone https://github.com/mohammaddSalim/Cinema-app.git
cd Cinema-app

# Install dependencies
npm install

# Run locally
npm run dev

# Build for production
npm run build
```

## Deployment

This project is deployed on **Vercel**. Any push to the `main` branch triggers an automatic redeployment.
